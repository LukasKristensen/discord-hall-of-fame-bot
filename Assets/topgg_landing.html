Hall of Fame Bot 🏆✨

Capture legendary moments in your Discord server!

Hall of Fame Bot automatically tracks the most reacted messages and immortalizes them on a special board once they surpass a set reaction threshold. Unlike traditional starboard bots, this bot works with all emojis, making every reaction count!

![test](https://raw.githubusercontent.com/LukasKristensen/discord-hall-of-fame-bot/refs/heads/main/Assets/main_context_2.png)

🌟 Features

  • ✅ Multi-Emoji Support – Not limited to one emoji like starboard bots.
  • ✅ Includes images/videos from messages.
  • ✅ Includes replies to conversation for full context.
  • ✅ Custom Reaction Thresholds – Set the number of reactions needed for a post to be featured.
  • ✅ Hall of Fame Board – Messages are automatically reposted once they reach the threshold.

How to Configure

By default, the reaction threshold is set to 7 reactions for a post to reach the Hall of Fame.

Use the command /reaction_threshold_configure to set a custom threshold for your server.

🔒 Custom Emoji Whitelist

By default all emojis count, but a custom emoji filter can be applied to the bot so that it will only look for emojis configured to the server.

  • To enable it: /custom_emoji_check_logic
  • Adding emojis to the whitelist: /whitelist_emoji
  • Removing emojis from the whitelist: /unwhitelist_emoji or /clear_whitelist

💬 Command List

  • /help – List of available commands and their descriptions.
  • /set_reaction_threshold <reaction_threshold:int> – Set the amount of reactions needed for a post to reach the Hall of Fame.
  • /include_authors_reaction – Should the author of a message be included in the reaction count?
  • /allow_messages_in_hof_channel – Allow anyone to type in the Hall of Fame channel.
  • /custom_emoji_check_logic <All emojis | Only whitelisted emojis> – Use only whitelisted emojis for the reaction count.
  • /whitelist_emoji <emoji> – Add a whitelisted emoji to the list (used with /custom_emoji_check_logic).
  • /unwhitelist_emoji <emoji> – Remove a whitelisted emoji from the list (used with /custom_emoji_check_logic).
  • /clear_whitelist – Clear the whitelist of emojis (used with /custom_emoji_check_logic).
  • /get_server_config – Get the current bot configuration for the server.
